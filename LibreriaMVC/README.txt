Programa Java que simula una librería.

El programa utiliza JavaFX y el patrón de diseño MVC (Modelo Vista Controlador), la vista es controlada por archivos FXML.


  Forma de compilar el programa:
        
    Utilizar la siguiente línea de comando. (Se utiliza ANT) 
       $ant
            
  Ejecutar el programa:

    Utilizar la siguiente línea de comando.
       $ java -jar libreriaMVC.jar
